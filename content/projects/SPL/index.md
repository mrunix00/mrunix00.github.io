[SPL (Simple Programming Language)](https://github.com/mrunix00/SPL) is a WIP interpreted, statically typed and VM-based programming language.

## Roadmap
- [x] Lexer
- [x] Parser
- [x] REPL shell
- [x] Variables
- [x] Functions
- [x] Conditions
- [x] While loops
- [x] For loops
- [ ] Arrays
- [ ] Strings
- [ ] Hashmaps
- [ ] Error handling
- [ ] Structs
- [ ] I/O operations
- [x] Bytecode VM
- [ ] Tail call optimization
- [ ] Garbage collector
- [ ] Importing libraries
- [ ] Standard library
- [ ] JIT compiler
